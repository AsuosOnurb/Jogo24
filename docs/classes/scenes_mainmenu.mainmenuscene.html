Class MainMenuScene

The class that models the main menu scene. This is the first scene that the player sees when he starts the game.

We're talking about a phaser scene, so every single method and function can be private.

Important: This object should not be bothered about handling player data, login things, backend things, or other logic related stuff.

  • update(time: number, delta: number): void

  • Inherited from Phaser.Scene.update

    • Defined in node_modules/phaser/types/phaser.d.ts:86215

    This method should be overridden by your own Scenes.

    This method is called once per game step while the scene is running.

    Parameters

    • time: number

      The current time. Either a High Resolution Timer value if it comes from Request Animation Frame, or Date.now if using SetTimeout.

    • delta: number

      The delta time in ms since the last frame. This is a smoothed and capped value based on the FPS rate.
