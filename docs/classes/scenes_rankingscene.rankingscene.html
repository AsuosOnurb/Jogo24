    The init() procedure provided by phaser.

    Here's where we setup the whole UI.

    It begins with one decision:

    We try to connect to the database to retrieve it's data. If there's no internet connection, or if the connection fails (for some reason), then we'll just load the blue background with an empty table.

    If the connection to the database is successfull, then things run normally.
