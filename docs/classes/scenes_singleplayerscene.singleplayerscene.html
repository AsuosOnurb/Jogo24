Class SingleplayerScene

Models the screen/scene where the singleplayer game takes place. This class/file is only responsible for the visual/interaction side of things. The game data/logics is handled in the SingleplayerGame.

Private HandleButtonClick_Number

  • HandleButtonClick_Number(clickedButtonIndex: number): void
    • Defined in src/scripts/scenes/SingleplayerScene.ts:389

    Handles the functionality of the number buttons on the card.

    remarks

    The notion of button index is very important in this game mode due to the Undo feature. In order to know what buttons need to change, each operation has 2 button indexes:

    1. The index for the first operand button
    2. A second index for the second operand button

    Because each card has 4 button, the index range is [0,1,2,3]. Index 0 relates to the left-most button. Index 1 relates to the top-most button. Index 2 relates to the right-most button. Index 3 relates to the bottom-most button.

    Parameters

    • clickedButtonIndex: number

      The index of the button that was clicked.

    Returns void
